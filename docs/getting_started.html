Getting Started with the HoneyComb SDK

Welcome to the Getting Started guide for using the Hoco Unity SDK. This page will help you understand the basics and get up and running quickly.

Quick Start

Follow these simple steps to begin:

ðŸ›  Installation

To use the HoneyComb SDK you need to install a few dependencies:

  • UniTask - Asynchronous Operations
  • Newtonsoft.Json - Cause Unity's Json serializer is bad mkay

Step 1: UniTask - Async Made Easy

‍Note: Not everyone needs UniTask, however it is light and efficient enough to be use by this SDK by default, If you really know what your doing you can always take out all the UniTask calls and put in native async or your own way of handling async calls. For more info about UniTask you can visit the Git Repo Here

UniTask is your go-to for handling asynchronous operations in Unity. It's efficient and more straightforward than Coroutines. Here's how to bring it into your project:

  1. Open Unity's PackageManager.
  2. Click on "Add package from git URL."
  3. Enter the following URL:
  • Import from PackageManager: Pop open your PackageManager and paste this into the 'Add package from git URL' field:
https://github.com/Cysharp/UniTask.git?path=src/UniTask/Assets/Plugins/UniTask

Step 2: NewtonSoft.Json - Transform Your Data

NewtonSoft.Json is fantastic for managing JSON data in Unity. Add it to handle data like a pro:

  • Import from PackageManager: Add by Package Name
    com.unity.nuget.newtonsoft-json

There you have it! Your Unity project is now equipped with some powerful tools. Ready to configure your project next!
